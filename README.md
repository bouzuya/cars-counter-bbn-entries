# cars-counter-bbn-entries

A [bouzuya/cars][] counter for entries in blog.bouzuya.net

## Installation

```bash
npm install @bouzuya/cars-counter-bbn-entries
```

## Configuration

```bash
export BBN_ENTRIES_TAGS='["diary"]'
```

See: [`_env`](_env).
